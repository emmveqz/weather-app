# Neudesic Weather


### Instructions

 - Install the npm packages accordingly:  
   `npm install`
 - Build the front-end app using:  
   `npm run build-ui`
 - Build the back-end app using:  
   `npm run server-ui`
 - Start the app:  
   `npm run start-server`
 - Access the app in your browser with the url:  
   `http://localhost:8181`
